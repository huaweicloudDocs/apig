# 查看专享版实例创建进度<a name="ZH-CN_TOPIC_0000001111862092"></a>

## 功能介绍

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=ShowDetailsOfInstanceProgressV2)中调试该接口。
