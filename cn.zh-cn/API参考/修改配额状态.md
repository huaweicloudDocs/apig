# 修改配额状态<a name="apig-phapi-180713107"></a>

## 功能介绍<a name="section52318866"></a>

本接口面向SC开放。当租户购买的API分组调用次数用尽时，SC调用此接口冻结购买者的调用；当购买者续费后，SC调用此接口解冻购买者的调用。
