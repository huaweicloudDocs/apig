# 查询签名密钥列表<a name="ZH-CN_TOPIC_0000001158502005"></a>

## 功能介绍

查询所有签名密钥的信息。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=ListSignatureKeysV2)中调试该接口。

## URI

GET /v2/\{project\_id\}/apigw/instances/\{instance\_id\}/signs
