# 修改分组下指定错误类型的自定义响应<a name="ZH-CN_TOPIC_0000001158502043"></a>

## 功能介绍

修改分组下指定错误类型的自定义响应。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=UpdateGatewayResponseTypeV2)中调试该接口。

## URI

PUT /v2/\{project\_id\}/apigw/instances/\{instance\_id\}/api-groups/\{group\_id\}/gateway-responses/\{response\_id\}/\{response\_type\}

## 请求示例

```
{
  "body" : "body: \"{\"error_code\":\"$context.error.code\",\"error_msg\":\"$context.error.message\",\"request_id\":\"$context.requestId\"}\"",
  "status" : 403
}
```
