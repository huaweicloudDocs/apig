# 导入API<a name="ZH-CN_TOPIC_0000001158502035"></a>

## 功能介绍

导入API。导入文件内容需要符合swagger标准规范，API网关自定义扩展字段请参考《API网关开发指南》的“导入导出API：扩展定义”章节。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=ImportApiDefinitionsV2)中调试该接口。

## URI

POST /v2/\{project\_id\}/apigw/instances/\{instance\_id\}/openapi/import
