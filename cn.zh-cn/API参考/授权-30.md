# 授权<a name="apig-phapi-180713046"></a>

## 功能介绍<a name="section18879496"></a>

APP创建成功后，还不能访问API，如果想要访问某个环境上的API，需要将该API在该环境上授权给APP。授权成功后，APP即可访问该环境上的这个API。
