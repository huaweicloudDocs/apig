# 查看ACL策略未绑定的API列表<a name="ZH-CN_TOPIC_0000001158502031"></a>

## 功能介绍

查看ACL策略未绑定的API列表，需要API已发布

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=ListApisUnbindedToAclPolicyV2)中调试该接口。

## URI
