# 查询环境列表<a name="ZH-CN_TOPIC_0000001158661981"></a>

## 功能介绍

查询符合条件的环境列表。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=ListEnvironmentsV2)中调试该接口。

## URI

GET /v2/\{project\_id\}/apigw/instances/\{instance\_id\}/envs
