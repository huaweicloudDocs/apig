# 自动生成APP Code<a name="ZH-CN_TOPIC_0000001158502021"></a>

## 功能介绍

创建App Code时，可以不指定具体值，由后台自动生成随机字符串填充。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=CreateAppCodeAutoV2)中调试该接口。

## URI

PUT /v2/\{project\_id\}/apigw/instances/\{instance\_id\}/apps/\{app\_id\}/app-codes
