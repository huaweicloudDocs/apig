# 查询资源概况<a name="apig-phapi-180713151"></a>

## 功能介绍<a name="section142594180538"></a>

提供给apiGateway\_admin角色的管理员的查询接口。

支持查询整体的资源使用情况。

包括租户数量\(提供者维度\)、租户数量\(使用者维度\)、API分组总数量、已上架API分组数量、APP总数量、云市场创建的APP数量、API总数量。
