# 调试API<a name="ZH-CN_TOPIC_0000001158661989"></a>

## 功能介绍

调试一个API在指定运行环境下的定义，接口调用者需要具有操作该API的权限。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=DebugApiV2)中调试该接口。

## URI

POST /v2/\{project\_id\}/apigw/instances/\{instance\_id\}/apis/debug/\{api\_id\}
