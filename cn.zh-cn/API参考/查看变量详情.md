# 查看变量详情<a name="ZH-CN_TOPIC_0000001158501993"></a>

## 功能介绍

查看指定的环境变量的详情。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=ShowDetailsOfEnvironmentVariableV2)中调试该接口。

## URI

GET /v2/\{project\_id\}/apigw/instances/\{instance\_id\}/env-variables/\{env\_variable\_id\}
