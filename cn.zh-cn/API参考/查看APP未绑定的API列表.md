# 查看APP未绑定的API列表<a name="ZH-CN_TOPIC_0000001111862058"></a>

## 功能介绍

查询指定环境上某个APP未绑定的API列表，包括自有API和从云市场购买的API。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=ListApisUnbindedToAppV2)中调试该接口。

## URI

GET /v2/\{project\_id\}/apigw/instances/\{instance\_id\}/app-auths/unbinded-apis
