# 创建VPC通道<a name="ZH-CN_TOPIC_0000001158662025"></a>

## 功能介绍

在API网关中创建连接私有VPC资源的通道，并在创建API时将后端节点配置为使用这些VPC通道，以便API网关直接访问私有VPC资源。

>![](public_sys-resources/icon-note.gif) **说明：** 
>每个用户最多创建30个VPC通道。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=CreateVpcChannelV2)中调试该接口。

## URI

POST /v2/\{project\_id\}/apigw/instances/\{instance\_id\}/vpc-channels
