# 查看域名证书<a name="ZH-CN_TOPIC_0000001158662015"></a>

## 功能介绍

查看域名下绑定的证书详情。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=ShowDetailsOfDomainNameCertificateV2)中调试该接口。

## URI

GET /v2/\{project\_id\}/apigw/instances/\{instance\_id\}/api-groups/\{group\_id\}/domains/\{domain\_id\}/certificate/\{certificate\_id\}
