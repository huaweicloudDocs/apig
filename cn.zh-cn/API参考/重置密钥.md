# 重置密钥<a name="ZH-CN_TOPIC_0000001111862062"></a>

## 功能介绍

重置指定APP的密钥。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=ResettingAppSecretV2)中调试该接口。

## URI

PUT /v2/\{project\_id\}/apigw/instances/\{instance\_id\}/apps/secret/\{app\_id\}
