## 功能介绍

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=AddingBackendInstancesV2)中调试该接口。

## URI

## 请求示例

-   VPC通道添加后端服务器地址

    ```
    {
      "members" : [ {
        "host" : "192.168.2.25",
        "weight" : 1
      } ]
    }
    ```

-   VPC通道添加云服务器

    ```
    {
      "members" : [ {
        "ecs_id" : "1082720c-3c15-409c-9ae3-4983ddfb6a9d",
        "ecs_name" : "APIGtest02",
        "weight" : 2
      } ]
    }
    ```
