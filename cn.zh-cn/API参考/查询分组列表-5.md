# 查询分组列表<a name="apig-phapi-180713022"></a>

## 功能介绍<a name="section46949754"></a>

查询API分组列表。

如果是租户操作，则查询该租户下所有的分组；如果是管理员操作，则查询的是所有租户的分组。
