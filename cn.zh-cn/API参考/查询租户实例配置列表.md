# 查询租户实例配置列表<a name="ZH-CN_TOPIC_0000001111862090"></a>

## 功能介绍

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=ListInstanceCofigsV2)中调试该接口。

## URI
