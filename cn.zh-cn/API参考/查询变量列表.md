# 查询变量列表<a name="ZH-CN_TOPIC_0000001158661983"></a>

## 功能介绍

查询分组下的所有环境变量的列表。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=ListEnvironmentVariablesV2)中调试该接口。

## URI

GET /v2/\{project\_id\}/apigw/instances/\{instance\_id\}/env-variables
