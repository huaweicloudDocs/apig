# 查看API绑定的签名密钥列表<a name="ZH-CN_TOPIC_0000001111862050"></a>

## 功能介绍

查询某个API绑定的签名密钥列表。每个API在每个环境上应该最多只会绑定一个签名密钥。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=ListSignatureKeysBindedToApiV2)中调试该接口。

## URI

GET /v2/\{project\_id\}/apigw/instances/\{instance\_id\}/sign-bindings/binded-signs
