# 查看自定义认证详情<a name="ZH-CN_TOPIC_0000001158662023"></a>

## 功能介绍

查看自定义认证详情

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=ShowDetailsOfCustomAuthorizersV2)中调试该接口。

## URI
