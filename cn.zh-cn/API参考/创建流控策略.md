# 创建流控策略<a name="apig-zh-api-180713064"></a>

## 功能介绍<a name="section20764951"></a>

当API上线后，系统会默认给每个API提供一个流控策略，API提供者可以根据自身API的服务能力及负载情况变更这个流控策略。

流控策略即限制API在一定长度的时间内，能够允许被访问的最大次数。
