# 查看后端实例列表<a name="ZH-CN_TOPIC_0000001111862082"></a>

## 功能介绍

查看指定VPC通道的弹性云服务器列表。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=ListBackendInstancesV2)中调试该接口。

## URI

GET /v2/\{project\_id\}/apigw/instances/\{instance\_id\}/vpc-channels/\{vpc\_channel\_id\}/members
