# 删除后端实例<a name="ZH-CN_TOPIC_0000001158502039"></a>

## 功能介绍

删除指定VPC通道中的弹性云服务器

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=DeleteBackendInstanceV2)中调试该接口。

## URI

DELETE /v2/\{project\_id\}/apigw/instances/\{instance\_id\}/vpc-channels/\{vpc\_channel\_id\}/members/\{member\_id\}
