# 查看特殊设置列表<a name="ZH-CN_TOPIC_0000001111862056"></a>

## 功能介绍

查看给流控策略设置的特殊配置。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=ListSpecialThrottlingConfigurationsV2)中调试该接口。

## URI

GET /v2/\{project\_id\}/apigw/instances/\{instance\_id\}/throttles/\{throttle\_id\}/throttle-specials
