# 解除访问限制<a name="apig-phapi-180713020"></a>

## 功能介绍<a name="section13874345"></a>

解除分组的访问限制。解除后，不再限制外部请求的次数，但是每个API会受到系统默认流控策略限制或受用户自定义的流控策略限制。
