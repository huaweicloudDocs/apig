# APP授权<a name="ZH-CN_TOPIC_0000001158502013"></a>

## 功能介绍

APP创建成功后，还不能访问API，如果想要访问某个环境上的API，需要将该API在该环境上授权给APP。授权成功后，APP即可访问该环境上的这个API。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=CreateAuthorizingAppsV2)中调试该接口。

## URI

POST /v2/\{project\_id\}/apigw/instances/\{instance\_id\}/app-auths
