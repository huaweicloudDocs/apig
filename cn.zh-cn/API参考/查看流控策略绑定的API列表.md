# 查看流控策略绑定的API列表<a name="ZH-CN_TOPIC_0000001158502009"></a>

## 功能介绍

查询某个流控策略上已经绑定的API列表。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=ListApisBindedToRequestThrottlingPolicyV2)中调试该接口。

## URI

GET /v2/\{project\_id\}/apigw/instances/\{instance\_id\}/throttle-bindings/binded-apis
