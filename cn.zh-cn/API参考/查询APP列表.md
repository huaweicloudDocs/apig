# 查询APP列表<a name="ZH-CN_TOPIC_0000001111702168"></a>

## 功能介绍

查询APP列表。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=ListAppsV2)中调试该接口。

## URI
