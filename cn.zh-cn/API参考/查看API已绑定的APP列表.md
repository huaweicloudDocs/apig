# 查看API已绑定的APP列表<a name="ZH-CN_TOPIC_0000001158502015"></a>

## 功能介绍

查询API绑定的APP列表。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=ListAppsBindedToApiV2)中调试该接口。

## URI
