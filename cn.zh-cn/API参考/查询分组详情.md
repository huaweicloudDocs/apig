# 查询分组详情<a name="ZH-CN_TOPIC_0000001158501989"></a>

## 功能介绍

查询指定分组的详细信息。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=ShowDetailsOfApiGroupV2)中调试该接口。

## URI

GET /v2/\{project\_id\}/apigw/instances/\{instance\_id\}/api-groups/\{group\_id\}
