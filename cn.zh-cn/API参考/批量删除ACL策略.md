# 批量删除ACL策略<a name="ZH-CN_TOPIC_0000001158662017"></a>

## 功能介绍

批量删除指定的多个ACL策略。

删除ACL策略时，如果存在ACL策略与API绑定关系，则无法删除。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=BatchDeleteAclV2)中调试该接口。

## URI

PUT /v2/\{project\_id\}/apigw/instances/\{instance\_id\}/acls
