# 修改特殊设置<a name="ZH-CN_TOPIC_0000001158662001"></a>

## 功能介绍

修改某个流控策略下的某个特殊设置。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=UpdateSpecialThrottlingConfigurationV2)中调试该接口。

## URI

PUT /v2/\{project\_id\}/apigw/instances/\{instance\_id\}/throttles/\{throttle\_id\}/throttle-specials/\{strategy\_id\}
