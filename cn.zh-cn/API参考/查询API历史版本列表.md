# 查询API历史版本列表<a name="ZH-CN_TOPIC_0000001111862044"></a>

## 功能介绍

查询某个API的历史版本。每个API在一个环境上最多存在10个历史版本。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=APIG&api=ListApiVersionsV2)中调试该接口。

## URI

GET /v2/\{project\_id\}/apigw/instances/\{instance\_id\}/apis/publish/\{api\_id\}
